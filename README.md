# 심리 신호 스캐너 (Stock Chart Analyzer) v1

차트(가격·거래량)만으로 **"지금 이 종목에서 군중이 탐욕인지 공포인지"** 를 읽고,
매수·매도·관망 신호와 **손절 기준**을 함께 제시하는 도구.

> 차트는 군중 심리의 지도이지 예언서가 아니다.
> 모든 신호는 **리스크 관리(손절)와 세트**로 동작한다.

## 무엇을 보나
- 대상: 종목 10개(고정 리스트, `config.py`) — 현재 5개 확정 + 5개 공란
- 데이터: **FinanceDataReader**(무료, API 키 불필요) 일봉 → 주/월봉 리샘플
- 산출물: 종목별 **신호 카드**(판정 + 근거 + 박스권/방어선 + 손절가) · CSV

## 분석 모듈 (OHLCV만으로 계산)
| 모듈 | 역할 | 점수 |
|---|---|---|
| ADX | 시장 국면(추세장/횡보장) 게이트 | — (가중치) |
| MA·다중TF | 추세·정배열·골든크로스·월주일 정렬 | −2~+2 |
| RSI | 모멘텀(과매수/과매도, 역추세) | −2~+2 |
| 지지/저항·박스·방어선 | 박스권·이탈금지선·POC | −2~+2 |
| 거래대금 | 위치×캔들로 진짜/가짜 돌파 | −2~+2 |
| **추세선** | 하락추세선/상승추세선 검출·돌파·전환 포착 | −2~+2 |
| ATR | 손절가·비중 자동 산출 | — (리스크) |

**지지/저항 산출 기준**(min/max를 넘어): 스윙 피벗 군집(터치 횟수×최근성 강도) ·
피보나치 되돌림 · 라운드넘버 · 매물대(POC·VAH·VAL).

**추세선**: 낮아지는 고점을 이어 하락추세선을, 높아지는 저점을 이어 상승추세선을
자동 검출. *하락추세 지속이면 매수 신호를 막고(veto)*, **하락추세선 임박(끝나감)·
상향돌파(추세 전환)** 종목을 포착한다. (`config.DOWNTREND_VETO`)

**매물대 심리**: 거래량 분포로 평단가를 근사 — 단기(3개월)/장기(1년) 평단을 나누고,
거래량 가중 **추정 평단가 대비 미실현손익**과 **머리 위 물량 비율**(현재가 위에 물린
거래량)을 계산해 "위가 무거운지/가벼운지"를 보여준다. *실제 개인 평단가는 사적
데이터라 못 구하므로 OHLCV 기반 근사임.*

→ 국면 가중치 적용 후 **−100~+100 정규화 점수**로 종합. (상세: `docs/설계결정_v2.md`)

## 설치
```bash
pip install -r requirements.txt
```
> 지표(ADX/RSI/ATR)는 외부 라이브러리 없이 pandas로 직접 계산한다.

## 실행
```bash
python main.py                 # config.STOCKS 전체 스캔(실데이터, 네트워크 필요)
python main.py --csv out.csv   # 결과 CSV 저장
python main.py --chart         # 종목별 차트(HTML) — 박스권·방어선·진입/손절 선
python main.py --chart --png   # 차트를 PNG로도 저장(kaleido+chrome 필요)
python main.py --dashboard     # 5종목 단일 페이지 대시보드(토글·전환후보 분류)
python main.py --backtest      # 차트 전용 리스크 관리 백테스트(R 단위 기대값·최대낙폭)
python main.py --demo --chart  # 합성 데이터로 데모(네트워크 불필요)

# 로컬 증분 캐시(data_cache/, gitignored, 종목당 ~30KB)
python main.py --backfill 200  # 캐시 없는 종목을 최대 200개 전체이력 백필(하루 N개씩 분산)
python main.py --update        # 캐시된 전 종목 증분 갱신(신규 봉만 — 매일 1회)
python main.py --cache --dashboard  # 캐시 기반으로 빠르게 분석/대시보드
```

증분 캐시: 전체 이력은 **1회 백필**(`--backfill N`을 며칠 나눠 실행), 이후 **`--update`로 신규 봉만** 받는다.
용량은 압축 gzip-CSV로 100종목 ≈ 3MB · 1,000종목 ≈ 30MB · 10,000종목 ≈ 300MB.

`--dashboard`는 전환후보/관망/회피로 묶은 종목 탭 + 지표 ON/OFF 토글 버튼이 있는
`dashboard.html`을 만든다. 호버는 마우스에 가장 가까운 한 요소만 떠 정보가 쏟아지지 않는다.

`--backtest`는 이 도구를 '예측'이 아니라 '리스크 관리'로 쓸 때의 성과를 과거 데이터로
검증한다. 라이브와 동일한 `analyze()`를 시점마다 재호출해 신호를 재현하고, 손익을
**R(리스크 단위)** 로 본다(+1R=손절폭만큼 이익, −1R=손절). 승률보다 기대값·최대낙폭으로 읽는다.

차트(`--chart`)는 캔들 위에 박스권 음영·핵심방어선·저항·POC·이평선과
진입/손절/목표 선을 그리고, **핵심 선에 마우스를 올리면 용어 설명**이 뜬다.
`charts/<code>.html`로 저장되며 브라우저에서 열면 인터랙티브하게 확대·툴팁 확인 가능.

`--demo`는 네트워크가 막힌 환경에서 파이프라인 로직을 확인하기 위한 모드다.
실데이터 신호를 보려면 FinanceDataReader가 외부(야후/KRX)에 접속 가능한 환경에서 실행해야 한다.

## 출력 예시
```
[종목명 CODE]                       심리: 🟢 강세 (정규화 +58점)
────────────────────────────────────────────────────────────
국면      : 추세장 (ADX 31)  방향 상승
추세      : 정배열 / 다중TF 월↑·주↑·일↑  [+2]
모멘텀    : RSI 58 (중립)  [+0]
지지/저항 : 방어선 지지 반등  [+2]
거래대금  : 거래대금 급증 + 양봉 → 진짜 돌파 (평균 대비 2.3배)  [+2]
────────────────────────────────────────────────────────────
박스권    : 76,000 ~ 82,000  (현재 79,200, 상단 −3.4%)
핵심방어선 : 76,000 ⚠️ 이탈금지  (신뢰도 강함, POC·MA120 겹침)
            └ 종가 이탈 시 추세 훼손 → 즉시 손절/회피
────────────────────────────────────────────────────────────
판정      : 저항 돌파 시 매수
손절      : 76,400  (ATR손절 / 방어선 중 택)
1차 목표  : 88,000  (손익비 1:2)
비중      : N주  (계좌의 1% 손실 허용 기준)
```

## 구조
```
config.py            전역 설정(종목·임계값·가중치·리스크)
glossary.py          용어 사전(카드 각주 + v1.5 툴팁 공용)
scanner/
  data.py            수집 + 리샘플
  indicators.py      ADX·MA·RSI·ATR·지지저항·박스·POC·거래대금
  levels.py          수평 지지/저항(스윙군집·피보·라운드·밸류영역)
  trendlines.py      하락/상승 추세선 검출·돌파·전환 + 하락추세 veto
  supply.py          기간분리 매물대(단기/장기 평단) + 미실현손익 추정
  scoring.py         국면 가중치 → 100점 정규화 → 판정
  analyze.py         종목 1개 분석 파이프라인
  card.py            신호 카드 렌더 + CSV
  chart.py           차트 시각화(plotly) + 용어 마우스오버
tests/sample_data.py 오프라인 데모용 합성 OHLCV
main.py              CLI
docs/                기획·설계·결정 문서
```

## 로드맵

- **v1.5** (현재): 박스권·방어선·신호 차트 시각화(plotly) + 용어 마우스오버 ✅

## 한계 및 주의 (정직 고지)
- 차트는 **군중 심리의 기록**이지 미래 보장이 아니다. 외부 충격(악재·실적·금리)엔 깨진다.
- 펀더멘털(회사 실적)은 차트에 없다 → 심리만 본다.
- 본 도구는 개인 판단 보조용이며 **투자 권유가 아니다**. 타인에게 신호 제공/판매 시
  자본시장법(유사투자자문업) 검토 필요.
- 최종 매매 책임은 사용자 본인에게 있으며, **손절 규칙 준수가 전제**다.
