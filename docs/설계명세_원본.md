# 심리 신호 스캐너 — 설계 명세서 (v1)

> 차트(가격·거래량)만으로 "지금 이 종목에서 군중이 탐욕인지 공포인지"를 읽고,
> 매수·매도·관망 신호와 **손절 기준**을 함께 제시하는 도구.
> 차트는 군중 심리의 지도이지 예언서가 아니므로, 모든 신호는 **리스크 관리(손절)와 세트**로 동작한다.

---

## 0. 개요

| 항목 | 내용 |
|---|---|
| 목적 | 차트만 보고 매매 판단을 돕는 신호 스캐너 |
| 대상 | 종목 **10개** (고정 리스트) |
| 봉(주기) | **일봉 / 주봉 / 월봉 / 년봉** (실시간 X) |
| 데이터 소스 | **FinanceDataReader** (API 키 없음·무료·요청제한 사실상 없음) |
| 산출물 | 종목별 **신호 카드**(판정 + 근거 + 손절가) |
| 비범위 | 펀더멘털(재무·뉴스·공시), 실시간 체결, 자동주문 |

### 데이터 처리 원칙
- 일봉을 FinanceDataReader로 수집 → 주/월/년봉은 코드에서 **리샘플**(OHLC 집계, V 합산).
- 모든 지표는 **OHLCV만으로 계산** (외부 지표 데이터 구매 불필요).

## 1. 지표 모듈 명세

각 모듈은 `점수(-2 ~ +2)`와 `근거 문구`를 반환한다. 점수는 §2 종합 로직에서 합산된다.

### 1-1. 시장 국면 — ADX  ⭐최우선
지표의 신뢰도를 좌우하는 **전제 조건**. 국면에 따라 어떤 신호를 믿을지 결정한다.

| 값 | 국면 | 신뢰할 지표 |
|---|---|---|
| `ADX > 25` | **추세장** | 이평선·돌파 (RSI 과매수는 무시) |
| `ADX < 20` | **횡보장** | RSI·지지/저항 반등 (돌파는 속임수 의심) |
| `20~25` | 전환 구간 | 관망 가중 |

- 계산: 표준 ADX(14). 라이브러리 `ta` 사용.
- 출력: `국면` 플래그(추세/횡보/중립) → 다른 모듈 점수에 **가중치**로 작용.

### 1-2. 추세 — 이동평균선 + 다중 시간프레임  ⭐
- MA 5 / 20 / 60 / 120 (종가 단순이평).
- **정배열**(5>20>60>120) = +2, **역배열** = -2.
- **골든크로스**(20선이 60선 상향돌파) = 매수 가점 / **데드크로스** = 감점.
- **다중 시간프레임 정렬**: 월·주·일봉 각각의 추세 방향(20선 기울기/배열)을 구해
  - 셋 다 상승 정렬 → **최강 매수(+2)**
  - 엇갈림 → 0, 셋 다 하락 → -2.
- 심리: 이평선 = 평균 매수단가. 위면 보유 안심, 아래면 투매 불안.

### 1-3. 모멘텀 — RSI(14)
- `RSI ≥ 70` 과매수(탐욕), `RSI ≤ 30` 과매도(공포).
- **국면 연동**: 추세장에선 과매수를 매도신호로 쓰지 않음(추세 지속). 횡보장에서만 역추세 신호로 사용.
- (Phase 2) RSI 다이버전스 → 단기고점/바닥 경고.

### 1-4. 지지 / 저항 + 매물대(POC)
- 수평 지지/저항: `rolling(20).min / max`, `rolling(60).min / max`.
- 라운드 넘버(심리적 가격대) 자동 표기.
- 피보나치 되돌림 38.2 / 50 / 61.8%.
- **매물대(Volume Profile)**: 가격을 20구간으로 나눠 구간별 거래량 합산 → **POC(거래량 최대 가격대) = 핵심 지지/저항**.
- **돌파 처리**: 저항을 거래량 동반 돌파 시 → 해당 선이 **지지로 역할 전환**.
- 심리: 본전심리·후회. 물린 사람이 많은 가격이 벽이 된다.

### 1-5. 거래량 / 거래대금 급증  ⭐(사용자 강조)
- **거래대금 = 종가 × 거래량** 사용(저가주 왜곡 제거, 거래량보다 정확).
- 급증 기준: `당일 거래대금 > 최근 20일 평균 × 2`.
- **위치 × 캔들 결합으로 의미 확정**:

| 위치 | 캔들 | 해석 | 점수 |
|---|---|---|---|
| 저항 돌파 | 양봉 | 진짜 돌파, 매수세 유입 | +2 |
| 저항 돌파 | 거래 평범 | 가짜 돌파 의심 | 0 |
| 바닥권 | 양봉 폭발 | 셀링클라이맥스(항복) | +1 |
| 고점권 | 장대음봉 폭발 | 물량 출회, 단기고점 | -2 |
| 신고가 | 거래 감소 | 상승 동력 약화 | -1 |

### 1-6. 변동성 — ATR (손절·비중)  ⭐
- ATR(14)로 종목별 평균 변동폭 측정.
- **손절가 = 진입가 − (2 × ATR)** (종목 변동성에 맞춰 자동).
- **비중(포지션) = 감당할 손실금액 ÷ (진입가 − 손절가)**.
- 손익비(R:R) **최소 1:2** — 손절폭의 2배 이상 위에 1차 목표가 설정.
- 심리/원칙: 변동성 큰 종목은 손절폭 넓게, 작은 종목은 좁게. 종목마다 같은 % 손절은 틀림.

### 1-7. (Phase 2) 일목균형표 — 구름대
- 전환선(9) / 기준선(26) / 선행스팬1·2(구름대, 26 앞) / 후행스팬(26 뒤).
- 주가가 **구름대 위 = 강세, 아래 = 약세**, 구름 두께 = 지지/저항 강도.
- v1의 MA·지지저항과 역할이 겹치므로 **v1 이후 별도 모듈로 추가**.

### 1-8. (Phase 2 후보) 상대강도 · 차트패턴
- 상대강도: 지수(코스피/S&P) 대비 수익률 → 10종목 중 주도주 선별.
- 차트패턴: 깃발·삼각수렴·헤드앤숄더 등 패턴 인식(구현 난이도 높음).

## 2. 종합 신호 로직

```
1) 국면 판정(ADX)  →  추세장 / 횡보장
2) 국면에 맞는 모듈만 가중치 부여
3) 각 모듈 점수(-2~+2) 합산  →  총점
4) 총점 → 판정 + 심리 게이지
```

| 총점 | 판정 | 심리 게이지 |
|---|---|---|
| `+4 이상` | 적극 매수 | 🟢 강세 |
| `+1 ~ +3` | 매수 관심 | 🟢 |
| `-1 ~ +1` | 관망 | ⚪ 중립 |
| `-3 ~ -1` | 매도 관심 | 🔴 |
| `-4 이하` | 적극 매도/회피 | 🔴 공포 |

> 게이지는 §0 컨셉("탐욕 vs 공포")을 한눈에 보여주는 요약 표시.

## 3. 출력 형식 — 신호 카드(예시)

```
[삼성전자 005930]                              심리: 🟢 강세 (총점 +5)
─────────────────────────────────────────────
국면      : 추세장 (ADX 31)
추세      : 월·주·일 모두 정배열 ✅  / 20선 골든크로스
모멘텀    : RSI 58 (중립, 추세장이라 과매수 여유)
지지/저항 : 지지 76,000 · 저항 82,000 (POC 80,500)
거래대금  : 평균 대비 2.3배 급증 + 양봉 → 저항 돌파 신호
─────────────────────────────────────────────
판정      : 저항 82,000 돌파 시 매수
손절      : 76,400 (진입 −2·ATR)
1차 목표  : 88,000  (손익비 1:2.1)
```

## 5. 한계 및 주의 (정직 고지)

- 차트는 **군중 심리의 기록**이지 미래 보장이 아니다. 외부 충격(악재·실적쇼크·금리)에 깨진다.
- 펀더멘털(회사 실적)은 차트에 없다 → 심리만 본다.
- 모든 신호는 **백테스트로 검증** 후 신뢰. 과최적화(curve fitting) 경계.
- 본 도구는 개인 판단 보조용이며 **투자 권유가 아니다**. 타인에게 신호 제공/판매 시 자본시장법(유사투자자문업) 검토 필요.
- 최종 매매 책임은 사용자 본인에게 있으며, **손절 규칙 준수가 전제**다.
