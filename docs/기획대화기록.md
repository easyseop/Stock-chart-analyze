# 주식 차트 분석기 — 기획 대화 기록 & 결정사항

> 토스증권 차트분석기처럼, **차트(가격·거래량)만 보고 매매를 돕는 도구**를 만들기 위한
> 기획 단계 논의 정리. (날짜: 2026-06-20)

---

## 1. 만들려는 것 (컨셉)

- 토스증권 차트분석기와 유사하되, **오직 차트만** 보고 매매 판단을 돕는 도구.
- 핵심 철학: **차트 = 군중 심리의 지도.** 지표는 그 심리를 읽는 번역기.
  단, 지도이지 예언서가 아니므로 **항상 리스크 관리(손절)와 세트**로 동작.
- 결론: **기술적으로 충분히 구현 가능.** 개인용이면 무료 데이터로 가능.

---

## 2. 확정된 결정사항

| 항목 | 결정 |
|---|---|
| 데이터 소스 | **FinanceDataReader** (API 키 없음·무료·요청제한 사실상 없음) |
| 종목 수 | **10개** (고정) |
| 봉(주기) | **일 / 주 / 월 / 년봉** (실시간 제외 — 너무 무거움) |
| 데이터 원재료 | **OHLCV** (시·고·저·종·거래량) — 모든 지표는 여기서 계산 |
| 산출물 | 종목별 **신호 카드**(판정 + 근거 + 손절가) |

### 왜 이 데이터인가
- 모든 기술적 지표의 원재료는 OHLCV 하나뿐. 별도 "지표 데이터"를 살 필요 없이 **계산으로 다 나온다.**
- 일봉으로 받아서 주·월·년봉은 코드에서 리샘플(집계)하면 됨.

---

## 3. 핵심 인사이트 — "지표는 사람 심리를 따른다"

차트의 가격·거래량은 결국 **매수자(탐욕)와 매도자(공포)가 부딪힌 결과**.
지표는 그 군중 심리를 숫자로 바꾼 것.

| 지표 | 깔려있는 심리 |
|---|---|
| 지지/저항·매물대 | 본전심리·후회 ("그 가격 오면 팔자/살걸") |
| 이동평균선 | 평균 매수단가 — 위면 안심, 아래면 투매 불안 |
| RSI 과매수/과매도 | 탐욕(70↑) / 공포(30↓)의 극단 |
| 거래량 폭발 | 관심·확신의 크기 (고점=차익실현, 바닥=항복) |
| 골든크로스 | 군중이 같은 신호 보고 동시 매수 → 자기실현 |

**왜 어느 정도 맞나:** ① 인간 심리는 잘 안 변해 패턴이 반복됨 ② 다수가 같은 선을 봐서 **자기실현적 예언**이 됨.
**한계:** 외부 충격(악재·실적·금리)엔 깨지고, 펀더멘털은 안 보임 → **그래서 손절이 필수.**

## 4. 사용할 지표 (중요도 × 구현 난이도 순)

### 가장 중요 + 쉬운 3종 (기본)
1. **이동평균선(MA) + 골든/데드크로스** — 추세 판단의 기본, 계산 한 줄
2. **지지/저항(전고점·전저점)** — 매수·손절 기준점
3. **RSI** — 단기 과열/침체, 매도 타이밍

### 거래량·매물대 (사용자 강조)
4. **거래대금 급증** (거래량×가격, 20일 평균 ×2) — "위치 × 캔들"로 진짜/가짜 돌파 구분
5. **매물대(Volume Profile)·POC** — 거래 몰린 가격 = 핵심 지지/저항

### 놓치기 쉬운데 중요해서 추가한 것
6. **시장 국면(ADX)** — 추세장/횡보장 구분. 어떤 지표를 믿을지 결정하는 전제 ⭐
7. **다중 시간프레임 정렬(월·주·일)** — 받아둔 데이터 활용, 최강 매수자리 탐지 ⭐
8. **ATR 손절·비중** — 종목 변동성에 맞춘 자동 손절가 + 손익비 1:2 ⭐

### Phase 2로 미룬 것
- **일목균형표(구름대)** — MA·지지저항과 역할 겹쳐 v1 신호 충돌 방지 위해 보류
- **상대강도(지수 대비), 차트패턴(깃발·헤드앤숄더 등)** — 구현 난도 높음

---

## 5. 매매·리스크 관리 기준

- **진입**: 지지선 반등 / 저항 돌파(거래량 동반) / 깃발·눌림목 재돌파
- **손절(택1)**: ① 지지선 이탈 ② 진입가 − 2×ATR ③ 고정 -5~8%
- **익절**: 다음 저항 / 트레일링 스탑 / **손익비 최소 1:2**
- **단기고점 신호(2개↑ 겹치면 경계)**: 전고점 부근 · RSI 70↑ 다이버전스 · 신고가인데 거래량 감소 · 윗꼬리 긴 캔들/장대음봉 · 볼린저 상단 이탈

## 7. 주의 (정직 고지)

- 차트는 심리의 기록이지 미래 보장이 아니다. 신호는 **백테스트로 검증** 후 신뢰.
- 본 도구는 개인 판단 보조용이며 **투자 권유 아님.** 타인에게 신호 제공/판매 시 자본시장법(유사투자자문업) 검토 필요.
- 최종 책임은 사용자 본인. **손절 규칙 준수가 전제.**
