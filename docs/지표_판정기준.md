## 0. 데이터
- 일봉 OHLCV 수집(미국=야후, 한국=야후 `.KS`/`.KQ`), 시작 `2019-01-01`.
- **주봉(W-FRI)·월봉(ME)은 일봉을 리샘플**: 시가=첫, 고가=max, 저가=min, 종가=마지막, 거래량=합.
- **미완성 봉 제외**(`DROP_UNCLOSED_BAR=True`): 이번 주/달의 진행 중 봉은 버려서 신호가 매일 흔들리지 않게.
  → 일봉 지표는 매일, 주봉은 주마감, 월봉은 월마감 때만 갱신된다.

## 1. 시장 국면 — ADX(14)  *(점수 아님, 게이트)*
표준 ADX. Wilder 평활.
- **ADX ≥ 25 → 추세장**(방향 있음)
- **ADX < 20 → 횡보장**(박스)
- **20 ~ 25 → 전환**(국면 바뀌는 중)
- 방향: `+DI ≥ −DI` 이면 상승, 아니면 하락 (세기와 별개의 방향 표시용).
- 용도: 국면에 따라 아래 모듈들의 **가중치를 다르게** 준다(§10).

## 2. 추세 — 이동평균 배열 + 크로스 + 다중 시간프레임  *(−2~+2)*
- **MA 배열**(일봉 5·20·60·120):
  - 정배열(단기>장기 순) → 기본 +2 / 역배열 → −2 / 혼조 → 0
- **골든/데드크로스**: MA20 vs MA60, **최근 5봉 내 실제 교차**만 인정 → +1 / −1 보너스
- **다중 시간프레임 방향**(`_tf_direction`): 각 프레임에서
  - **종가 > 대표MA 그리고 그 MA가 상승 중 → +1(↑)**
  - **종가 < 대표MA 그리고 그 MA가 하락 중 → −1(↓)**
  - 그 외(엇갈림) → 0(→)
  - 대표MA: **월=12개월, 주=20주, 일=20일**
  - 배열이 혼조(0)일 때, 월·주·일 셋 다 +1이면 +2 / 셋 다 −1이면 −2로 방향 결정
- 최종 점수 = (배열 ± 크로스) 를 −2~+2로 클램프.
- 카드 표기 예: `정배열 / 다중TF 월↑·주→·일↓ / 골든크로스`

## 3. 모멘텀 — RSI(14)  *(−2~+2, 역추세)*
- RSI ≤ 30 → **+2** (과매도/공포 → 반등 기대)
- RSI ≤ 40 → +1 (약세권)
- 40 ~ 60 → 0 (중립)
- RSI ≥ 60 → −1 (강세권/과열 주의)
- RSI ≥ 70 → **−2** (과매수/탐욕 → 단기조정 경계)
- ※ 횡보장에서 가중치 최대, **추세장에선 가중치 0**(추세장의 단독 역추세는 위험).

## 3ب. 상대강도(RS) · 신고가 · 시장방향  ★모멘텀 핵심 (비중 상향)
> 리서치상 실증 근거가 가장 강한 모멘텀/추세추종 계열. RSI·추세선보다 비중을 높였다.
- **상대강도(RS)** *(−2~+2)*: 종목/지수 비율 라인. 최근 60거래일 동안 **지수 대비 상대수익차(rel)**:
  - 지수(미국=S&P500, 한국=KOSPI) 대비 +10%↑ → **+2**, +3%↑ → +1, ±3% → 0, −3%↓ → −1, −10%↓ → **−2**
  - "시장보다 강한 종목이 주도주" — 가장 비중 큰 모듈.
- **신고가(52주)** *(−2~+2)*: 최근 252일 고가 대비 위치:
  - 고가 ±2% 이내(신고가권) → **+2**, −10% 이내 → +1, 중간 → 0, −25%↓ → −1, −50%↓ → **−2**
  - 신고가권은 머리 위 매물이 없어 상승이 가볍다.
- **시장방향** *(−2~+2)*: 지수가 50일선 위·상승 → +2 / 위 → +1 / 아래 → −1 / 아래·하락 → −2.
  - 개별 종목도 시장을 크게 타므로 하락장에선 매수 비중을 낮춘다.

## 4. 지지/저항 · 박스권 · 방어선  *(−2~+2)*
- **박스 탐지**: 후보 구간 20·60봉 중 (폭 < 15%인 '좁은 박스' 우선 → 경계 터치 밀도 높은 쪽).
  경계는 **직전까지**로 산출(현재 봉 제외) → 현 시점의 이탈을 잡기 위함.
- **현재 위치(position)와 점수**(근접 판정 ±2%):
  - 박스하단 −2% 밑 이탈 → **−2** (박스이탈, 방어선 붕괴)
  - 박스하단 ±2% + 당일 양봉 → **+2** (바닥권 지지 반등)
  - 박스상단 +2% 위 돌파 → **+2** (저항 돌파, 상단이 지지로 전환)
  - 박스상단 ±2% 임박(미돌파) → **−1** (매도벽 임박)
  - 그 외 → 0 (중단)
- **핵심 방어선 = 박스 하단**. 신뢰도 = 하단 ±2% 안에 `POC·MA60·MA120·라운드넘버`가 몇 개 겹치나:
  2개↑ 강함 / 1개 보통 / 0개 약함.
- **라운드넘버**: 자릿수 단위(10ⁿ)와 그 절반(5×10ⁿ⁻¹)에 ±0.5% 이내일 때만 인정(예: 75,000·80,000 O).

## 5. 거래대금(턴오버)  *(−2~+2)*
- 턴오버 = 종가 × 거래량. **20일 평균 대비 배수(mult)** 로 본다. 급증 기준 = **2배 이상**.
- 점수(박스 position·캔들과 조합):
  - 급증 + (저항돌파/박스이탈) + 양봉 → **+2** (거래 동반 진짜 돌파)
  - 급증 + 바닥권 + 양봉 → +1 (셀링클라이맥스/항복)
  - 급증 + 고점권 + 장대음봉(몸통≥3%) → **−2** (물량 출회)
  - 돌파인데 거래 미동반 → 0 (가짜 돌파 의심)
  - 그 외 → 0

## 6. 추세선(사선)  *(−2~+2)*  ★전환 후보 핵심
- **스윙 피벗**: 좌우 3봉(`SWING_K=3`)의 극값을 고점(H)/저점(L)으로. 탐색 180봉.
- **하락추세선** = 최근 '낮아지는 고점' 2점 연결 / **상승추세선** = '높아지는 저점' 2점 연결. 근접·터치 ±2%.
- **상태 판정(하락선 우선)**:
  | 상태 | 점수 | 조건 |
  |---|---|---|
  | **추세 전환 확정** | +2 | 하락선 상향돌파 + **N봉(기본3) 안착** + **상승추세선(저점↑) 형성** + **거래량 ≥1.5배** |
  | 돌파 후 횡보·전환대기 | +1 | 돌파 + N봉 안착(상승추세선 아직 / 또는 거래량 미동반) |
  | 하락추세선 돌파(미확인) | 0 | 갓 돌파 — 되밀림 위험, 안착 전 |
  | 하락추세선 임박 | +1 | 선 바로 아래 ±2%(끝나갈 가능성) |
  | 하락추세 지속 | −2 | 선 아래(고점 낮아짐) → **매수 veto** |
  | 상승추세 유지 | +1 | 상승선 위 |
  | 상승추세선 이탈 | −2 | 상승선 −2% 하향 이탈 |
- **거래량 필터**(백테스트 검증): 거래량 미동반 돌파는 가짜가 많아 → '전환 확정'을 '전환 대기'로 격하.
- 차트엔 **현재→미래 점선 연장**(추세 지속 시 선이 갈 자리)도 그린다.

## 7. 수평 지지/저항 레벨 (차트용, 점수 아님)
- **스윙 피벗 군집**: ±1.5% 내 피벗을 묶어 '강도점수'(터치 수·근접도). 최소 2터치.
- **피보나치 되돌림**: 0.236·0.382·0.5·0.618·0.786.
- **밸류영역(매물대 70%)**: VAL(하단)·POC(중심)·VAH(상단). POC=거래 최다 가격대.

## 8. 매물대 심리 (정보 게이지, 점수 미반영)
- **기간분리 평단**(거래량 분포 = 평단가 분포 근사): 단기=최근 63일, 장기=최근 252일.
- **추정 평단가** = 거래량 가중 평균가. **미실현손익% = (현재가−추정평단)/추정평단**.
  - +10% 밖 → 평균 이익(차익실현 압력) / −10% 밖 → 평균 손실(본전 저항) / 사이 → 중립.
- **머리 위 물량**(현재가보다 위에 쌓인 거래량 비율): ≥50% 무거움(저항 큼) / ≤25% 가벼움.

## 9. 점수 종합 → 판정
- 기여 모듈 **8개**: trend · **rs · newhigh · market** · volume · sr · rsi · trendline (각 −2~+2).
  *(ADX=게이트, ATR=리스크용 → 제외)*
- **국면별 가중치**(추세장/횡보장/전환) — 방침: RS·신고가·시장방향·거래량 ↑ / RSI·추세선 ↓:
  | 모듈 | 추세장 | 횡보장 | 전환 |
  |---|---|---|---|
  | trend | 1.0 | 0.5 | 0.5 |
  | **rs(상대강도)** | **1.2** | **1.0** | **1.2** |
  | **newhigh(신고가)** | 1.0 | 0.6 | 0.8 |
  | **market(시장방향)** | 1.0 | 0.8 | 0.8 |
  | volume(거래량) | 1.0 | 0.8 | 0.8 |
  | sr | 0.5 | 1.0 | 0.5 |
  | rsi | 0.0 | **0.4** | **0.2** |
  | trendline | **0.4** | **0.2** | 0.5 |
- **정규화점수** = 100 × Σ(점수×가중치) / Σ(2×가중치) → **−100 ~ +100**.
- **판정 임계값(대칭)**: |점수| ≥ **50** 적극, ≥ **20** 관심, 그 사이 관망.
  - +50↑ 적극매수 / +20↑ 매수관심 / ±20 관망 / −20↓ 매도관심 / −50↓ 적극매도·회피
- **하락추세 veto**(원칙): '하락추세 지속'이면 매수 신호를 막고 🔴 회피로 표기.

## 10. 리스크 / 포지션
- **진입(기준가)**: 보통 현재가. 단 고점권(저항 임박)이면 '박스 상단 돌파가'.
  → 매수 명령이 아니라, 이 가격을 기준으로 손절·목표·수량을 계산하는 **기준점**.
- **손절**: ATR손절(진입 − 2×ATR) 과 방어선손절(방어선 −1%) 중 `nearest`(진입에 가까운 쪽). 진입보다 최소 1% 아래로 보정.
- **1차 목표**: 진입 + (손익비 2 × 손절폭) → **손익비 1:2**.
- **비중**: 계좌의 **1% 손실**만 허용 → 수량 = (계좌×1%) ÷ 손절폭. (가상계좌: USD 1만 / KRW 1천만)

## 11. 전환 후보 분류(대시보드 좌측 버킷)
- 🟢 **전환 후보**: 추세 전환 확정 / 돌파 후 횡보·전환대기 / 하락추세선 임박
- 🔴 **회피**: 하락추세 지속 / 상승추세선 이탈
- ⚪ **관망**: 그 외(돌파 미확인 포함)

### 솔직한 한계 (백테스트·리서치 기준)
- 차트 신호의 **방향 예측력은 사실상 0**(무차별 매매 승률 ≈ 33% = 1:2 손익분기). 추세선·패턴은 엣지 약함.
- 실증 근거가 강한 건 **변동성 기반 비중·모멘텀(상대강도)** 뿐. 이 도구는 '예측기'가 아니라
  **손절·비중·손익비로 손실을 −1R로 가두는 리스크 통제 장치**로 쓰는 게 정직하고 유효하다.
- '거래량 동반' 필터는 백테스트에서 전환 후보 기대값을 음(−)에서 양(+)으로 돌린 유일한 단일 필터.
