## 0. 새 세션에서 처음 할 일
1. 이 브랜치 체크아웃: `git checkout claude/korean-text-review-o3wmsv`
2. 이 문서(`docs/HANDOFF.md`)와 `docs/설계결정_v2.md`, `README.md` 읽기
3. (네트워크 열렸으면) `pip install -r requirements.txt && python main.py --csv result.csv` 로 실데이터 확인
4. (네트워크 막혔으면) `python main.py --demo` 로 합성 데이터 동작 확인

## 1. 프로젝트 한 줄 요약
차트(OHLCV)만으로 군중 심리(탐욕/공포)를 읽어 **매수·매도·관망 신호 + 손절가**를
종목별 **신호 카드(텍스트/CSV) + 차트(HTML/PNG)** 로 내는 스캐너. 종목 10개(현재 5개 확정).

## 2. 지금까지 만든 것 (완료)
- **v1 코어**: ADX 국면 · MA·다중TF 추세 · RSI · 지지저항/박스/방어선/POC · 거래대금 · ATR 손절·비중
- **점수 체계**: 5개 모듈(trend/rsi/sr/volume/trendline) → 국면 가중치 → **−100~+100 정규화** → 판정
- **코드리뷰 반영**: 10건 수정 (pos_high 오판, 크로스 가드, 리샘플 미완성봉, POC 분산, 라운드넘버, 표시 오도 등) — `git log` 참고
- **v1.5 차트(plotly)**: 캔들+박스권+방어선+POC+이평선+진입/손절/목표 + 마우스오버 용어 툴팁
- **지지/저항 다중기준** (`levels.py`): 스윙피벗 군집(강도점수)·피보나치·라운드넘버·밸류영역(VAH/VAL)
- **추세선** (`trendlines.py`): 하락/상승 추세선 자동검출, 상태(지속·임박·돌파), **하락추세 veto**,
  선의 현재 가격값(`trend_line`)·거리(`trend_dist_pct`)를 데이터로 노출
- **매물대 심리** (`supply.py`): 기간분리 평단(단기 3M/장기 1Y) + 미실현손익 추정(추정평단·손익%·머리위물량)
- 용어사전(`glossary.py`), `--demo`(오프라인 합성데이터), 5번째 데모 시나리오 `reversal`

## 3. 핵심 설계 결정 (요약 — 상세는 docs/설계결정_v2.md)
- 점수 기여 모듈 5개. ADX=게이트, ATR=리스크 계산용(점수 제외)
- 국면별 가중치 매트릭스 (`config.REGIME_WEIGHTS`)
- 시간프레임 = 월·주·일 (년봉 제외: 데이터 부족)
- 판정 임계값: |점수|≥50 적극, ≥20 관심 (`config.VERDICT_STRONG/WEAK`)
- **하락추세 veto**: 하락추세 지속이면 매수 신호 차단 (`config.DOWNTREND_VETO`, 사용자 원칙)
- 매물대 심리는 **점수 미반영**(지지저항과 중복 방지) — 정보 게이지로만 제공

## 4. 종목 (config.STOCKS) — 5개 확정 + 5개 공란
RDDT(레딧) · ENPH(엔페이즈) · HMC(혼다 ADR) · UPST(업스타트) · 006800(미래에셋증권)

## 5. 실데이터 1차 결과 해석 (참고)
- 🔴 회피(하락추세 지속): **UPST**(평균 −33%·머리위 77%), **HMC**(−8%·71%)
- 🟢 전환 포착: **ENPH**(하락추세선 막 돌파·머리위 9%), **006800**(하락추세선 임박+RSI 과매도반등)
- ⚪ 관망: **RDDT**(전 모듈 중립)
- 참고: 006800(미래에셋) 4.8만원대는 **정상** — 2025 한국 증권주 랠리 반영(1년새 2만→7만→4.8만 눌림)

## 6. 알아둘 한계 / 주의
- **이전 원격 환경은 네트워크 차단**으로 실데이터 불가였음 → 새 환경은 네트워크 허용 정책 필요(§아래)
- AI는 정확한 실시간 시세를 모름 → **이상치는 데이터 내부 일관성으로만 검증**(거래량0·가격 급점프·음수 등),
  AI 기억과 비교 금지
- `ta` 라이브러리 미사용(빌드 실패) → ADX/RSI/ATR은 pandas 직접 구현
- 차트가 선이 많아 다소 빽빽함(표시 토글 미구현)
